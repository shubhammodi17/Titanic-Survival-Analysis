# Titanic-Survival-Analysis

Project Overview

The Titanic dataset contains passenger details such as age, gender, ticket class, and survival status.  
The goal of this project is to perform **data cleaning, analysis, and visualization** to understand the factors affecting survival.

---

Objectives

- Load and explore the Titanic dataset
- Handle missing values (data cleaning)
- Analyze survival patterns
- Visualize insights using graphs

---

Tools & Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

Key Analysis Performed

### 1. Survival by Gender
- Females had a significantly higher survival rate than males

### 2. Survival by Passenger Class
- Passengers in **1st class** had higher survival chances
- Passengers in **3rd class** had the lowest survival rate

### 3. Survival by Age Group
- Children had better survival chances compared to adults

---

Visualizations Included

- Bar chart: Survival by Gender
- Bar chart: Survival by Passenger Class
- Bar chart: Survival by Age Group
- Histogram: Age distribution

---

Data Cleaning Steps

- Filled missing values in **Age** using median
- Filled missing values in **Embarked** using mode
- Dropped **Cabin** column due to excessive missing values

---

Key Insights

- Gender is a strong factor affecting survival
- Socio-economic status (class) influenced survival
- Younger passengers had higher survival priority
